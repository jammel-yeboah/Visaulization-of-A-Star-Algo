## Getting Started

### Prerequisites
* pip


1. Clone the repo
   ```sh
   git clone https://github.com/jammel-yeboah/Visaulization-of-A-Star-Algo
   ```
3. Install Python packages
   ```sh
   pip install pygame --upgrade
   ```
    ```sh
   pip install python-math
   ```

  ```sh
  python a_star.py
  ```
